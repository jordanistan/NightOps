# Austin Atlas Data

`austin-v1.csv` contains the two initial public observing fields documented by
the Austin Astronomical Society: Inks Lake State Park and the Pedernales Falls
State Park Star Theater. Coordinates and the reported Bortle 4.5 values are
source-grounded and should be reviewed before the catalog version changes.

Sources:

- https://austinastro.org/index.php/aas-observing-fields/
- https://tpwd.texas.gov/state-parks/parks/things-to-do/stargazing/bortle-ratings
